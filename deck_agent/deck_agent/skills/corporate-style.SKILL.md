# corporate-style skill (PLACEHOLDER)

> Replace with your real brand guidelines.

Governs ALL visual and formatting choices. The style tokens below are the ONLY
ones the renderer supports — never invent a token not listed here.

## Chart style tokens
- `corporate_ts`        — time-series line charts
- `corporate_bar`       — categorical bar charts
- `corporate_waterfall` — waterfall / bridge charts

## Table style tokens
- `risk_standard` — default risk-report table
- `risk_compact`  — tighter style for dense tables

## Text style tokens
- `body`, `callout`, `footnote`
